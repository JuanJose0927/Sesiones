Sesión 6


En esta sesión se aprendio a como crear una tabla en un codigo de html

Pasos a seguir Lo que se debe de realizar
Paso 1 Crea una nueva carpeta en el escritorio llamada tablas Posteriormente, arrastra la carpeta a Visual Studio Code y crea dentro de ella un archivo llamado tablas.html
Paso 2 Genera la estructura básica de html:5 , cambia el idioma y el título del documento será tablas. Pon un h1 de encabezado que diga: Tablas
Paso 3 Para hacer una tabla nosotros utilizamos la etiqueta (table-table)
Paso 4 Una tabla se compone de filas y columnas. Para hacer una fila pondremos la etiqueta tr. Haremos una tabla con dos filas por lo tanto pondremos dos tr. Recuerda que los tr se deben cerrar
Paso 5 Para colocar el contenido dentro de esa fila utilizaremos la etiqueta td. Así que dentro de cada fila pondremos de a td indicando que una es la primera fila y la otra la segunda fila
Paso 6 Agrega un segundo al cual pondrás de nombre seguna columna
Paso 7 Por lo tanto, cada tr es toda una fila y el contenido se coloca en columnas td
Paso 8 table border = "1" (Atributo para aplicar en la tabla)
